# 📘 A Tour Of Cpp

이 레포지토리는 **『 A Tour Of Cpp』** 도서를 기반으로  
C++을 체계적으로 학습하고 발표 자료와 예제 코드를 정리하는 **스터디 기록용 레포지토리**입니다.

매주 정해진 분량을 학습한 뒤,  
각자 맡은 챕터를 **정리 · 발표 · 코드 공유**하는 방식으로 진행합니다. 

---

## 📅 스터디 개요
- 주제: A Tour of C++ 책 기반 C++ 기초 및 핵심 개념 학습
- 참여자: 한지윤, 김완영, 남기운, 이솔, 윤상흠
- 기간: 2026년 6월 24일 ~ 2026년 7월 29일 (약 6주)
- 일정: 매주 수요일 오후 8시 30분 Teams 미팅
- 진행 방식:
  - 지정 분량 사전 학습 후 발표 및 토론
  - 핵심 개념 정리 및 실무 적용 가능성 공유
